NUSGo! is a hacking experiment that aims to solve a social problem. It is built during a 6 days extended hackathon by a group of NUS School of Computing students within the walls of Tembusu College, to vent some post-exam frustration (and also because we like to hack).

NUSGo! is currently in its beta testing phase and more features/perks (ie. discount from NUS restaurants) may be introduced, depending on the number of users during this testing period.

Hopefully, NUSGo! will be fully released by August 2015!

Features (Beta V1.0):

  • Real-time map updates
  • Facebook integration for easy login
  • Instant messaging and notifications
  • Search for places of interests
  • Open chat environments to promote interaction
  • Support for multiple chat rooms
  • Cute original emojis

Tools/Frameworks/APIs:

  • Node.js
  • Socket.io
  • jQuery
  • Postgre SQL
  • Google Maps API
  • Facebook API
