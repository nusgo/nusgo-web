NUSGo! is a hacking experiment that aims to solve a social problem. It is built during a 5 days extended hackathon by a group of NUS School of Computing students, within the walls of Tembusu College.

NUSGo! is currently in its beta testing phase and more features/perks (ie. discount from NUS restaurants) may be introduced, depending on the number of users during this testing period.

Hopefully, NUSGo! will be fully released by August 2015!

Features (Beta V1.0):

  • Real-time map updates
  • Facebook integration for easy login
  • Instant messaging and notifications
  • Search for places of interests
  • Open chat environments to promote interaction
  • Responsive layout for mobile devices
  • Cute emojis by Team RollingGO!

Tools/Frameworks/APIs:

  • Node.js
  • Socket.io
  • jQuery
  • Postgre SQL
  • Google Maps API
  • Facebook API

Looking for meal buddies?


Place a marker on the map to show that you are up for a meal!

Or you can click on a marker to join him/her for a meal!


You will be notified when you receive meal requests.


NUSGo! is a web application for lonely hearts to look for meal buddies. We believe that nobody deserves to eat alone.
